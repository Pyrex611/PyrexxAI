Pyrexx AI — Next.js + TypeScript + Tailwind + Framer Motion demo website

Quick start
1. Install dependencies
   npm install

2. Run dev server
   npm run dev

3. Open http://localhost:3000

Features
- Next.js App Router (TypeScript)
- Tailwind CSS for styling
- Framer Motion for smooth animations
- Responsive, sticky navbar, CTA sections
- Mock API endpoint: POST /api/book

Notes
- Replace placeholder assets in /public with brand assets.
- Hook /api/book to your calendar/CRM in production.
- Add analytics, A/B testing, and server-side link tracking for conversion optimization.
